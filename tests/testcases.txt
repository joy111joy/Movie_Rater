# Test Cases for Customer Management System

## UI Test Cases

1. As a customer, I can see a web page listing all the movies from the database.
2. As a staff member, I can see a web page that allows me to add a new movies to the database.
3. As a staff member, I can see a web page that allows me to update an existing movies's information.
4. As a staff member, I can see a web page that allows me to delete a movie from the database.

## API Test Cases

1. As a business partner, I can fetch all the movies from the database via a REST API, receiving JSON data.
2. As a business partner, I can add a new movie to the database via a REST API.
3. As a business partner, I can update an existing movie's information via a REST API.
4. As a business partner, I can delete a movie from the database via a REST API.
